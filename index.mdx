HashRails provides financial services API for payments, virtual accounts, and digital transactions, and this documentation covers all available endpoints with detailed examples and response schemas.

## Base URLs

- **Production**: `https://api.hashrails.xyz/`
- **Staging**: `https://api.railsfromthecrypt.com/`

## Rate Limiting

API requests are rate limited to ensure fair usage. Rate limits are applied per API key and reset every hour.

## Getting Started

1. Obtain your API key from the HashRails dashboard
2. Include the API key in the `x-api-key` header for all requests
3. Start making API calls to our endpoints

For detailed authentication instructions, see the [Authentication](/api-reference/authentication) section.
